# What is En Croissant?

En Croissant is a chess database and analysis tool. It's designed to be closer in user experience to online chess websites, while still providing all the powerful features only a desktop application can offer.

## Key Features

*   **Analyze**: Store and analyze your games from [lichess.org](https://lichess.org) and [chess.com](https://chess.com) with top engines.
*   **Database**: Manage your games with a powerful database system.
*   **Repertoire**: Prepare your openings with the integrated explorer.
*   **Easy Management**: Simple engine and database installation and management directly within the app.
